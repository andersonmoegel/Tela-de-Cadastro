# Electron Client Registration App

Este é um aplicativo Electron simples que fornece um formulário de cadastro de clientes.

## Funcionalidades

- Formulário de cadastro de clientes com campos para nome, email e telefone
- Validação dos campos de entrada
- Interface de usuário limpa e intuitiva

## Pré-requisitos

- [Node.js](https://nodejs.org/) (versão 12 ou superior)
- [npm](https://www.npmjs.com/) (gerenciador de pacotes do Node.js)

## Instalação

1. Clone este repositório:
   ```
   git clone https://github.com/andersonmoegel/Tela-de-Cadastro
   ```

2. Navegue até o diretório do projeto:
   ```
   cd Tela-de-Cadastro
   ```

3. Instale as dependências:
   ```
   npm install
   ```

## Execução

Para executar o aplicativo Electron, use o seguinte comando:

```
npm start
```

Isso irá iniciar o aplicativo e exibir a janela do formulário de cadastro de clientes.

## Personalização

Você pode personalizar o estilo e a funcionalidade do formulário de cadastro de clientes editando os arquivos `index.html` e `main.js`. Sinta-se à vontade para adicionar mais recursos, como conexão com um banco de dados ou envio de e-mails, de acordo com suas necessidades.
